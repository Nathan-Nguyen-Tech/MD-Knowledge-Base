## Card Naming Convention

**Format**: `DIAG_[SUBCLASS]_[NUMBER]`

**Subclass Abbreviations**:
- `CHEM` = Chemistry Panels
- `LIPID` = Lipid Testing
- `DM` = Diabetes Testing
- `THY` = Thyroid Testing
- `HEM` = Hematology
- `COAG` = Coagulation
- `CARD` = Cardiac Biomarkers
- `INFLAM` = Inflammatory Markers
- `VITMIN` = Vitamin/Mineral
- `HORM` = Hormone Testing
- `URINE` = Urine Tests
- `STOOL` = Stool Tests
- `ID` = Infectious Disease
- `HOME` = Home Monitoring

## Notes

1. **Fasting requirements**: Specify in cards (lipid panel no longer requires fasting per AHA)
2. **Specimen type**: Include collection instructions for home tests
3. **Reference ranges**: Include lab-specific ranges where applicable
4. **Trending**: Many tests are more valuable as trends than single values
5. **Not included**: Genetic testing, specialized immunology panels, cancer markers (oncology category)

---

## Clinical Pearls for Card Development

1. **A1c Cards**: Include eAG conversion, factors affecting accuracy (hemoglobinopathies)
2. **TSH Cards**: Include reference range changes in pregnancy, subclinical disease thresholds
3. **Lipid Cards**: Include non-HDL calculation, when to recheck
4. **Troponin Cards**: Include timing of draws, delta changes for diagnosis
5. **Home BP Cards**: Include proper technique, when to contact provider
6. **CGM Cards**: Include time in range goals, calibration requirements
