## Card Naming Convention

**Format**: `PROC_[TYPE]_[NUMBER]`

**Type Abbreviations**:
- `DX` = Diagnostic Procedures
- `TX` = Therapeutic Procedures
- `SURG` = Surgical Procedures
- `VAX` = Vaccinations
- `SCREEN` = Screening Procedures

---

## Notes

1. **Consent**: All procedures require informed consent - include in cards
2. **Prep instructions**: Include bowel prep (colonoscopy), fasting (EGD), medication holds
3. **Recovery**: Include post-procedure care, activity restrictions, warning signs
4. **Prior authorization**: Many procedures require PA - note in cards
5. **Surgical procedures**: Include as reference cards for patient education; actual ordering is specialist-directed

---

## Clinical Pearls for Card Development

1. **Colonoscopy Cards**: Include prep options, sedation types, polyp surveillance intervals
2. **Joint Injection Cards**: Include max frequency (3-4/year), post-injection care, steroid precautions
3. **Vaccine Cards**: Include schedules, contraindications, special populations, storage
4. **Stress Test Cards**: Include medication holds (beta blockers, caffeine), exercise alternatives
5. **Skin Biopsy Cards**: Include technique differences (punch vs shave vs excisional), wound care
