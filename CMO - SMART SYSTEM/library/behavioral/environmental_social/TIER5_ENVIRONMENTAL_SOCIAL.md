## Card Naming Convention

**Format**: `ENV_[CATEGORY]_[NUMBER]`

**Category Abbreviations**:
- `SOC` = Social/Community
- `REL` = Relationships
- `LONE` = Loneliness Interventions
- `AIR` = Air Quality
- `HOUSE` = Housing Safety
- `GREEN` = Green Space/Neighborhood
- `WORK` = Workplace Ergonomics
- `BALANCE` = Work-Life Balance
- `TOB` = Tobacco Cessation
- `ALC` = Alcohol
- `HARM` = Harm Reduction
- `FIN` = Financial/Resources

## Notes

1. **Social determinants**: These cards address SDOH - key to health outcomes
2. **Resource localization**: Cards should include how to find local resources
3. **Cultural sensitivity**: Community engagement varies by culture
4. **Privacy**: Substance use cards should note confidentiality
5. **Professional referral**: Include when to refer to social work, counseling

---

## Clinical Pearls for Card Development

1. **Social Isolation Cards**: Include screening tools (UCLA Loneliness Scale)
2. **Smoking Cessation Cards**: Include 5 A's framework, quit line numbers
3. **Ergonomics Cards**: Include visual guides, adjustment instructions
4. **Resource Cards**: Include national resources + how to find local ones
5. **Air Quality Cards**: Include AQI monitoring, when to limit outdoor activity
