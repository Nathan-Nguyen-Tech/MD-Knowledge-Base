## Card Naming Convention

**Format**: `MB_[CATEGORY]_[NUMBER]`

**Category Abbreviations**:
- `MIND` = Mindfulness Meditation
- `FOCUS` = Focused Attention
- `LK` = Loving-Kindness
- `MBSR` = Mindfulness Programs
- `YOGA` = Yoga Styles
- `YOGACOND` = Yoga for Conditions
- `TAI` = Tai Chi
- `QI` = Qigong
- `BIO` = Biofeedback/Self-Regulation

## Clinical Pearls for Card Development

1. **Meditation Cards**: Include "thoughts are not facts" concept, non-judgment emphasis
2. **Yoga Cards**: Include modifications for common limitations, when to avoid inversions
3. **Tai Chi Cards**: Include balance safety (near support), slow progression
4. **MBSR Cards**: Include 8-week program structure, daily practice requirements
5. **All Cards**: Include beginner-friendly entry points, progression pathway
