## Card Naming Convention

**Format**: `REC_[CATEGORY]_[NUMBER]`

**Category Abbreviations**:
- `SLEEP` = Sleep Schedule
- `ENV` = Sleep Environment
- `PRESLEEP` = Pre-Sleep Routines
- `CBTI` = CBT-I Techniques
- `COG` = Cognitive Techniques
- `BEH` = Behavioral Techniques
- `BREATH` = Breathing Techniques
- `RELAX` = Relaxation Techniques

---

## Notes

1. **CBT-I**: Gold standard for insomnia - cards should reference this
2. **Professional help**: Include when to seek help (persistent insomnia, severe anxiety)
3. **Medical causes**: Note when sleep/stress issues may indicate underlying condition
4. **Tracking**: Include sleep diary recommendations
5. **Consistency**: Emphasize that techniques require regular practice

---

## Clinical Pearls for Card Development

1. **Sleep Schedule Cards**: Include weekend consistency importance, jet lag tips
2. **Digital Sunset Cards**: Include blue light filter alternatives, audiobook suggestions
3. **Breathing Cards**: Include step-by-step instructions, practice frequency
4. **PMR Cards**: Include script or link to guided audio
5. **Stress Cards**: Include when stress becomes clinical anxiety requiring treatment

## Sleep Metrics Reference (Include in Cards)

| Metric | Normal Adult | Poor Sleep Indicator |
|--------|--------------|---------------------|
| Sleep Latency | <20 min | >30 min |
| Wake After Sleep Onset | <30 min | >60 min |
| Total Sleep Time | 7-9 hours | <6 hours |
| Sleep Efficiency | >85% | <75% |
