## Card Naming Convention

**Format**: `MOV_[CATEGORY]_[NUMBER]`

**Category Abbreviations**:
- `WALK` = Walking
- `RUN` = Running/Jogging
- `CYCLE` = Cycling
- `SWIM` = Swimming/Water
- `GROUP` = Group Fitness
- `BW` = Bodyweight
- `FW` = Free Weights
- `RB` = Resistance Bands
- `MACH` = Machines
- `FUNC` = Functional
- `STRETCH` = Static Stretching
- `DYN` = Dynamic Stretching
- `YOGAFLEX` = Yoga for Flexibility
- `BAL` = Static Balance
- `BALDY` = Dynamic Balance
- `FALL` = Fall Prevention

## Notes

1. **Medical clearance**: Note when provider clearance needed (cardiac history, recent surgery)
2. **Progression**: Include how to progress intensity safely
3. **RPE scale**: Include Rate of Perceived Exertion for intensity guidance
4. **FITT principle**: Include Frequency, Intensity, Time, Type for each card
5. **Contraindications**: Include when exercise should be modified or avoided

---

## Clinical Pearls for Card Development

1. **Walking Cards**: Include target heart rate zone, step tracking, proper footwear
2. **Strength Cards**: Include proper form, breathing, rest between sets
3. **Balance Cards**: Include safety (hold onto chair), progression steps
4. **All Cards**: Include warm-up/cool-down reminders, hydration

## Exercise Intensity Reference (Include in Cards)

| Intensity | % Max HR | RPE (0-10) | Talk Test |
|-----------|----------|------------|-----------|
| Light | 50-63% | 2-3 | Easy conversation |
| Moderate | 64-76% | 4-6 | Can talk, slight breathlessness |
| Vigorous | 77-93% | 7-8 | Difficult to talk |
