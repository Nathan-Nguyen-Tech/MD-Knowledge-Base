# TIER 4 OUTLINE - SMART Card Library

## Purpose

Comprehensive Tier 4 subcategories for all 79 Tier 3 categories.

**Key Principles**:
1. **Frequency/Duration = Template Field** (not separate cards)
2. **Device vs Application**: Equipment prescription (DME) vs. using equipment (Recovery/Treatment)
3. **Evidence-Based Only**: All interventions backed by research
4. **Hierarchy Endpoint**: [T4 = CARD] or [→ T5]
