# SMART DECK LIBRARY INDEX

## Purpose

The SMART Deck Library **replaces traditional patient education materials** (PDFs, printouts, pamphlets) with **actionable, progressive card sequences** organized by clinical condition.

**Key Innovation**: Instead of handing patients walls of text, clinicians prescribe **condition-specific decks** where education happens through doing.

## ğŸ¯ HOW CLINICIANS USE THIS LIBRARY

### Scenario 1: New Hypertension Diagnosis

**OLD WAY**:
```
1. Hand patient "Understanding Hypertension" PDF
2. Write prescription for Lisinopril
3. Tell them to "follow up in 4 weeks"
4. Hope they read the PDF (they don't)
```

**NEW WAY**:
```

2. Deck auto-publishes to patient's app
3. Patient sees Card 1 (micro-learning)
4. After completion, Card 2 unlocks (medication)
5. System tracks adherence, BP readings
6. Clinician reviews dashboard weekly
7. Patient learns by doing, step-by-step
```

- Sees progress (HbA1c trending down)
- Feels supported (engaged with care)
